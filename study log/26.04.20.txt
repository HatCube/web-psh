운영체제 주요 기능
1. 처리능력의 향상
> 주어진 시간 내의 얼마나 많은 일을 처리하는지

2.응답시간 단축
사용자가 요청한 작업에 대하여 결과가 나올때까지 걸리는 시간

3.신뢰도 향상
>시스템이 오류없이 안정적으로 동작하고 문제가 생겨도 잘 복구되도록 하는 것

4.인터페이스 제공
>사용자와 컴퓨터가 소통할 수 있게 해주는 접점

5.입출력 관리
>입력 장치와 출력장치가 정상적으로 동작하도록 관리

6.프로그램 및 사용자 보호
> 프로그램끼리 서로 함부로 침범하지 못하게 하고 사용자 데이터도 안전하게 지키는 기능

7.주기억 장치(RAM) 관리
필요한 프로그램에 메모리를 할당하고, 사용이 끝나면 회수

8.CPU관리
운영체제가 cpu를 어떤 프로그램에 먼저 사용할지 결정

9.파일관리
파일을 만들고 저장하고 삭제하고 찾고 보호하는 기능

10.명령어 해석 및 실행
사용자가 입력한 명령어를 해석하고 실행하는 기능

type : 파일에 들어있는 내용을 터미널에 출력하는 기능
>로그 확인
>설정파일 확인
>간단한 텍스트 내용 확인

type 파일명1,파일명2 : 여러개의 파일을 이어서 출력할 수 있다.

type 파일명1 > 파일명2 : 파일명1의 내용을 파일명2로 저장할 수 있다.

type 파일명1, 파일명2 > 파일명 3: 파일명1,2의 내용을 합쳐서 파일명3으로 저장할 수 있다.

type con : 키보드에서 값을 입력받아 출력
컨Z누르고 엔터치면 종료됨

type con > 파일명 : 키보드에서 값을 입력받아 파일명에 기록

파일만들때 사용하는 echo
echo 내용 > 파일명 : 파일을 만들고 안에 내용을 작성해준다.
