한 컴퓨터에서 인터넷 브라우저,문서 편집기,메신저가동시에 실행되고있다.

이 상황과 관현된 운영체제의 기능 2가지를 쓰고 이유를 설명하라

touch : 파일을 새로 만들거나, 기존 파일의 시간 정보를 변경할때 사용하는 명령어

access time (atime) : 파일을 읽은 시간
modification time(mtime) : 파일 내용을 수정한 시간
change time : 파일 상태 정보가 변경된 시간


touch -c 파일명 : 파일이 없으면 생성하지 마라
touch -t 날짜 파일명 : 내가 원하는 시간을지정할 수 있다.
>202604211030

touch -d "날짜" 파일명 : 사람이 읽기 쉬운 날짜 형식 사용
>2026-04-21 10:30:00

echo : 텍스트를 화면에 출력하거나 파일로 저장할 때 사용하는 기본적인 명령어

리눅스에서는 쉘(shell)이 변수를 관리한다.
변수 생성 > 쉘이 저장 > echo가 출력

*쉘(shell)*
사용자가 입력한 명령어를 운영체제에 전달해 실행하는 "명령어 해석기"

우리가 명령어를 입력하는 공간 > 터미널
shell은 터미널 안에서 동작하는 프로그램

Bash : 리눅스에서 자주 사용되는 쉘(shell)프로그램

윈도우에서도 shell이 있다.
1.command prompt(cmd)
>가장 오래된 windows 기본 shell

2.powerShell
>현재 windows의 기본 shell
>객체 기반 명령어 사용
>자동화,스크립트에 매우 강력

현재 터미널 세션에서만 사용 가능 > 터미널을 닫으면 변수는 사라진다.

echo는 파일 작업에서 가장 많이 사용된다.

파일 생성 및 덮어쓰기
echo "hello" > file.txt
>file.txt 생성
>기존 내용이 있다면 덮어쓴다.

파일의 추가

echo "World" >> file.txt
>기존내용 유지
>뒤에 추가로 작성

프로그램을 백그라운드에서 실행하기
>명령어를 실행하면서도 터미널을 계속 사용할 수 있게 하는 방법
포어그라운드(foreground) > 현재 터미널을 점유
백그라운드(background) 뒤에서 실행 (터미널 사용 가능)

기본적인 방법은 명령어  뒤에 & 붙이기
>sleep 100 & 
[1]12345
[1] > 작업번호
12345 > PID
