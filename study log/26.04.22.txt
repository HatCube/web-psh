패키지 관리자 
리눅스는 텍스트 기반이라 웹에 직접 접근할 수 없기 때문에 패키지 관리자가 저장소에 접근하여 프로그램을 직접 다운 받아 준다.
sudo apt install 프로그램명

배포(AWS를 이용한 클라우드 배포)

배포
내 컴퓨터에 서비스를 만든다.(로컬 환경 : 다른 사람들이 접속을 못함)
다른사람들이 내 IP주소를 모른다.(포트 설정)
외부로 노출이 되는거다 보니 보안과 네트워크에 대한 위험성이 존재함.

클라우드 배포
컴퓨터를 빌려서 내 서비스를 집어넣고 사람들이 접속하도록 하는것
원격 컴퓨터의 IP주소를 알려줘서 다른 사람들이 접속하게 하면 된다.

웝서버 WAS : xhazot, Nginx
sudo apt install nginx
