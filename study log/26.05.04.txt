OSI 7계층

> 데이터가 포장 -> 전달 -> 해석되는 과정

물리 계층
>데이터를 0과 1의 전기 신호로 변환하여 전달
프로토콜 없음
데이터 상태 : 비트(bit)

장비
>허브(hub) : 이곳에 연결된 장치 모두에게 데이터를 전달
리피터(repeater) : 신호를 증폭 시킨다.
케이블(Cable) : 데이터 이동 통로

데이터링크 계층
같은 네트워크 내부로 전달

MAC주소 기반으로 정확하게 전달
>네트워크에 연결할 수 있는 기기에 할당된 주소

데이터 상태 : 프레임(Frame)

장비
> 스위치(switch)
>브릿지(bridge)

네트워크 계층
> 목적지까지 가장 안전하고 빠르게 데이터를 보내는 계층
>최적의 경로를 설정해야 함 



미들웨어

정규화
>이상현상을 제거하기 위해 테이블을 분리하는 것
>삽입이상 : 데이터를 추가할 때 필요없는 내용까지 추가되는 것
>갱신이상 : 중복된 데이터가 있을때 일부분만 수정하는 것
>삭제 이상 : 데이터를 삭제할 때 필요한 내용까지 삭제되는 것

1차
>모든 속성이 원자값이어야 한다.

2차
>부분함수 종속을 해소

3차
> 이행함수 종속을 해소 

BCNF
>3차 정규화를 좀 더 엄격하게 관리(결정자는 반드시 후보키여야 한다)

4차
>관계없는 속성은 분리한다.

5차
>조인 종속을 제거한다.

SQL 함수
>다양한 기능들을 제공하는 함수

문자열 함수
숫자 함수
날짜 함수 

단일행 함수
>1개의 행에 대하여 함수를 적용하여 1개의 행을 반환
 
집계 함수
> 1개 이상의 행애 대해서 함수를 적용하여 1개의 값을 반환

count()
sum()
greatest()

>그룹별 : Group by (속성별로 집계함수를 적용 시킬수 있다.)
