데이터에 대해서 조회 수정 추가 삭제하는 명령어

select * from 테이블;

Like
>문자열 안에서 특정 글자나 패턴을 찾을때 사용하는 조건문
select 컬럼명
from 테이블명
where 컬럼명 like '패턴'

select * from film where title = 'love'
>이 SQL은 제목이 정확히 LOVE인 영화만 찾는다.

like는 포함 여부, 시작문자,끝문자,특정 글자 패턴을 찾을 수 있다.

% > 글자 수와 상관 없이 아무 문자나 여러개 올 수 있다는 뜻

'A%' > A로 시작하는 데이터
'%A' > A로 끝나는 데이터 
'%A%' > A를 포함하고 있는 데이터
'A%B' > A로 시작하고 B로 끝나는 값

_ > 문자 1개를 의미
'A_' > A로 시작하고 전체 길이가 2인 값을 의미
'A__' > A로 시작하고 전체 길이가 3인 값
'_A' > A로 끝나는 전체 길이가 2인 값
'_A_' > 전체 길이가 3이고 A가 가운데 있는 값

DCL(Data Control language) : 데이터 제어어
grant : 권한 부여
revoke : 권한 회수
>데이터베이스에서 사용자 권한을 관리하는 SQL이다.

DCL이 필요한 이유
>모든 사용자에게 모든 권한을 주면 위험해진다.

MySQL에서는 사용자를 다음과 같이 구분한다.
'사용자명'@'접속자위치'
'teacher'@'localhost'

사용자 생성하는법
create user '사용자명'@'접속자위치'
identified by '비밀번호';
